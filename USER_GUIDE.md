## Getting Started

### Creating an Account

1. Navigate to the AEGIS+ homepage
2. Click **"New Account"** or **"Register"**
3. Enter your **institutional email** (e.g., `student@institute.edu`)
4. Create a strong password
5. Select your role:
   - **Student**: For enrolled students
   - **Faculty**: For teaching staff
   - **Authority**: For administrative staff (requires approval)
6. Complete your profile information
7. Verify your email address

### Logging In

1. Go to the login page
2. Enter your email and password
3. Click **"Sign In"**
4. You'll be redirected to your role-specific dashboard

### Password Reset

1. Click **"Forgot Password?"** on the login page
2. Enter your registered email
3. Check your inbox for a password reset link
4. Follow the link and create a new password

## Student Guide

### Dashboard Overview

Your dashboard provides quick access to:
- **Grievances**: Submit and track issues
- **Resources**: Access study materials
- **Opportunities**: Browse internships and research positions
- **Scholar's Ledger**: Manage your tasks and goals

### Submitting a Grievance

1. Navigate to **Grievances > Submit New**
2. Fill in the form:
   - **Category**: Select the type (Infrastructure, Academic, etc.)
   - **Priority**: Choose urgency level
   - **Location**: Specify where the issue is
   - **Description**: Provide detailed information
   - **Photo** (optional): Upload supporting image
   - **Anonymous**: Toggle if you want to remain anonymous
3. Click **"Submit Grievance"**
4. Note your **Tracking ID** for future reference

### Tracking Grievances

1. Go to **Grievances > My Grievances**
2. View all your submitted grievances
3. Click on any grievance to see:
   - Current status
   - Timeline of updates
   - Authority remarks
   - Resolution details

### Accessing Academic Resources

1. Navigate to **Resources**
2. Use filters to find materials:
   - **Course**: Select your course
   - **Type**: Notes, Slides, Papers, etc.
   - **Tags**: Search by topic
3. Click **"Download"** to save resources
4. Use the search bar for quick access

### Browsing Opportunities

1. Go to **Opportunities > Browse**
2. Filter by:
   - Department
   - Type (Internship/Research)
   - Duration
   - Skills required
3. Click on an opportunity to view details
4. Click **"Apply"** to submit your application

### Applying for Opportunities

1. Open the opportunity details
2. Click **"Apply Now"**
3. Upload your:
   - **Resume** (PDF format recommended)
   - **Portfolio** (optional link)
   - **Cover Letter** (text)
4. Click **"Submit Application"**
5. Track status in **Opportunities > My Applications**

### Using Scholar's Ledger

1. Navigate to **Scholar's Ledger**
2. **Add Task**: Click **"+ New Task"**
   - Enter title and description
   - Set due date
   - Choose priority and category
3. **Update Progress**: Click on a task to update
4. **Mark Complete**: Check the box when done
5. **View Analytics**: See your productivity stats

## Faculty Guide

### Dashboard Overview

Your dashboard includes:
- **Upload Resources**: Share materials with students
- **Post Opportunities**: Create internship/research positions
- **Manage Applications**: Review student applications
- **Analytics**: Track resource downloads and engagement

### Uploading Academic Resources

1. Navigate to **Resources > Upload**
2. Fill in the details:
   - **Title**: Name of the resource
   - **Course**: Select the course
   - **Type**: Notes, Slides, Papers, etc.
   - **Description**: Brief overview
   - **Tags**: Add searchable keywords
3. Click **"Choose File"** to upload
4. Click **"Upload Resource"**
5. Resource is now available to students

### Managing Resources

1. Go to **Resources > My Uploads**
2. View all your uploaded resources
3. See download statistics
4. **Edit**: Update resource details
5. **Delete**: Remove outdated materials

### Posting Opportunities

1. Navigate to **Opportunities > Post New**
2. Fill in the form:
   - **Title**: Position name
   - **Type**: Internship/Research/Project
   - **Description**: Detailed information
   - **Required Skills**: List necessary skills
   - **Duration**: Expected timeframe
   - **Stipend**: Compensation details (if any)
   - **Deadline**: Application deadline
3. Click **"Post Opportunity"**

### Reviewing Applications

1. Go to **Opportunities > Applications**
2. View all applications for your opportunities
3. Click on an application to see:
   - Student profile
   - Resume and portfolio
   - Cover letter
4. Update status:
   - **Shortlisted**: Move to next round
   - **Accepted**: Offer the position
   - **Rejected**: Decline application
5. Add review notes (optional, internal)

### Managing Opportunities

1. Navigate to **Opportunities > My Posts**
2. View all your posted opportunities
3. **Edit**: Update opportunity details
4. **Close**: Stop accepting applications
5. **Mark as Filled**: When position is filled

## Authority Guide

### Dashboard Overview

Your dashboard provides:
- **Grievance Management**: Review and resolve issues
- **Department View**: Filter by your department
- **Analytics**: Track resolution metrics
- **Priority Queue**: See urgent grievances first

### Viewing Grievances

1. Navigate to **Grievances Dashboard**
2. Use filters:
   - **Status**: Submitted, In Progress, Resolved
   - **Category**: Infrastructure, Academic, etc.
   - **Priority**: High, Medium, Low
   - **Date Range**: Custom time period
3. Click on a grievance to view full details

### Updating Grievance Status

1. Open a grievance
2. Review the details and timeline
3. Click **"Update Status"**
4. Select new status:
   - **Under Review**: Acknowledged
   - **In Progress**: Being worked on
   - **Resolved**: Issue fixed
5. **Add Remark** (required): Explain the update
6. Click **"Save Update"**
7. Student receives automatic notification

### Adding Remarks

1. Open a grievance
2. Scroll to **"Add Remark"** section
3. Enter detailed notes about:
   - Actions taken
   - Expected timeline
   - Resources allocated
4. Attach supporting documents (optional)
5. Click **"Add Remark"**

### Assigning Grievances

1. Open a grievance
2. Click **"Assign"**
3. Select:
   - **Department**: Responsible department
   - **Authority**: Specific person (optional)
4. Add assignment notes
5. Click **"Assign Grievance"**

### Viewing Analytics

1. Navigate to **Analytics**
2. View metrics:
   - Total grievances by status
   - Average resolution time
   - Category breakdown
   - Priority distribution
3. Export reports as CSV/PDF

## Admin Guide

### Dashboard Overview

Your dashboard includes:
- **User Management**: Create and manage accounts
- **System Analytics**: Platform-wide metrics
- **Audit Logs**: Security and activity logs
- **Configuration**: System settings

### Managing Users

1. Navigate to **Admin > Users**
2. View all registered users
3. **Add User**: Click **"+ New User"**
   - Enter email and role
   - Send invitation
4. **Edit User**: Update profile or role
5. **Deactivate**: Disable account access
6. **Delete**: Permanently remove user

### Viewing Audit Logs

1. Go to **Admin > Audit Logs**
2. Filter by:
   - User
   - Action type
   - Date range
   - Resource type
3. Export logs for compliance

### System Analytics

1. Navigate to **Admin > Analytics**
2. View comprehensive metrics:
   - User distribution by role
   - Grievance statistics
   - Resource popularity
   - Opportunity engagement
   - Platform activity trends
3. Generate custom reports
4. Export data for analysis

### Platform Configuration

1. Go to **Admin > Settings**
2. Configure:
   - Email templates
   - Notification preferences
   - File upload limits
   - Retention policies
3. Save changes

## Common Features

### Profile Management

1. Click your avatar in the top-right
2. Select **"Profile"**
3. Update:
   - Personal information
   - Contact details
   - Profile picture
4. Click **"Save Changes"**

### Notifications

1. Click the bell icon in the header
2. View recent notifications
3. Click on a notification to view details
4. Mark as read or clear all

### Search Functionality

1. Use the search bar in the header
2. Search across:
   - Grievances (by ID or description)
   - Resources (by title or tags)
   - Opportunities (by title or skills)
3. Use filters to refine results

### Dark Mode

1. Click your avatar
2. Select **"Appearance"**
3. Toggle **"Dark Mode"**
4. Preference is saved automatically

### Getting Help

1. Click **"Help"** in the footer
2. Access:
   - FAQ
   - User guides
   - Contact support
   - Report a bug
